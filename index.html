Project Overview

EACTO is an advanced financial risk management system that leverages concepts from information theory, statistical modeling, and cybernetic control systems. The framework outperforms static risk models by dynamically adjusting risk thresholds - becoming more conservative during high uncertainty periods and more aggressive during predictable market conditions.

Market Complexity Analysis

Multiple entropy measures to quantify market uncertainty and identify regime changes in real-time.

Getting Started

Quick Setup

# Clone the repository
git clone https://github.com/Sakeeb91/EACTO.git
cd EACTO

# Install dependencies
pip install -r eacto_project/requirements.txt

# Generate sample data and run the system
cd eacto_project
python create_sample_data.py
python main.py

# Generate visualization dashboard
python generate_demo_results.py
